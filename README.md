This project integrates a **ChatGPT-like chat widget** into my personal portfolio site.

### What it does
- Chat UI with **thread history (left sidebar)** and **message view (right panel)**
- Stores chats in **localStorage** for persistence
- Calls a backend endpoint (Render) that connects to **OpenAI API**
- Built as a reusable widget that can be mounted on any page

### Tech Stack
- Frontend: React + TypeScript (Vite build)
- Backend: Node.js + Express (Render deployment)
- Model: OpenAI API (chat responses)
